# 🎓 Student Management System using MySQL

## 📖 Project Description

This project demonstrates SQL concepts using a Student Management Database. It includes table creation, joins, aggregate functions, views, indexes, stored procedures, and transaction control commands.

## 🚀 Features

- Database Creation
- Table Creation
- Primary Key
- Foreign Key
- Aggregate Functions
- Joins
- Subqueries
- Views
- Indexes
- Stored Procedures
- Transaction Control

## 🗄️ Database Tables

### Student Table
- StudentID
- StudentName
- Department
- Age

### Marks Table
- StudentID
- Subject
- Marks
